Welcome to DandyReads

DandyReads is a full stack web application built using Django and Python. Users can search for books on Google Books api and will be able to add them to their reading list. DandyReads also provides TIP (Time Investment Prediction) calculation for users to understand how long a certain book will take them to read based on the average reading speed and the size of the book.
